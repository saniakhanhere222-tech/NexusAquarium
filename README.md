# 🌊 Nexus Aquarium

Welcome to **Nexus Aquarium** – an immersive dark theme website experience that brings the wonders of the ocean to your screen.  

---

## 🎨 Theme & Design

- **Dark ocean theme** for a sleek, modern look.
- One animated and one static carousel for easy viewers interaction. 
- Smooth, responsive layout for desktop and mobile  
- Engaging **flip cards** to showcase marine life  
- Header that **turns transparent on scroll** for a dynamic effect  

---

## 🖼️ Features

- **Cool animated slider/carousel** for featured exhibits  
- Interactive gallery with high-quality underwater images  
- Upcoming events section with clear, organized layout  
- Responsive sections for medium large and smaller screen sizes.  

---

## 🚀 Technologies Used

- HTML5 & CSS3  
- JavaScript (Vanilla JS)  
- Flexbox & CSS Grid for responsive design  
- Font Awesome icons for stylish visuals
